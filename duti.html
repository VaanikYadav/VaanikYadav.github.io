Project Title

Duti FasTag

Platform

Native Mobile Application (iOS and Android)

Tags

Product Design

Toll Collection System

About

Duti is a mobile application for FASTag, a toll collection system in India that allows for cashless payment of toll charges. The project aims to create an intuitive and user-friendly interface that allows users to easily manage their FASTag accounts, make toll payments, and have a streamlined experience while using the application.

Ask

Design an application to manage FasTag toll collection

Must have:

🚀 Buy FasTag and manage vehicle

🚀 Add a wallet feature to pay tolls via FasTag

🚀 View transaction and travel history

Design Research

About the Research

To obtain crucial knowledge and comprehend the desires and requirements of our intended audience, I utilized a combination of methods, such as administering a questionnaire, executing a competitor evaluation, developing user profiles, and implementing empathy maps. This all-inclusive research technique helped us collect varied viewpoints and detect potential issues and possibilities in the toll payment app industry.

In summary, this extensive research methodology supplied us with invaluable information regarding our potential clientele's preferences and challenges when using toll-collection applications.

User Survey

About the Research

The results of this survey indicate that some users are unhappy with the overall design and user experience of the FasTag application, finding it challenging to navigate. However, a significant number of users encounter technical issues while using the app, and customer support is considered mediocre or unsatisfactory.

This feedback can be employed to improve the app and address the concerns raised by users.

Findings

Research Findings

Following extensive user research encompassing user surveys and competitive assessments, we discovered several crucial factors to consider during the ideation and design process.

User Requirements

Automatic deduction option should to be included for quicker toll transactions

Pain points

Using FasTag for toll transactions can be challenging, as there aren't any reliable applications

Must-have Requirements

Navigation should be seamless and incorporate one-tap functions in the application

Other Issues

90% of FasTag users feel dissatisfied when required to seek approval for payments while driving

Outcomes

Project Outcomes

This undertaking provided me with insightful viewpoints on toll gathering systems and highways. It presented a thrilling challenge that allowed me to address the requirements of daily FasTag consumers. The user interface garnered praise from users, owing to an in-depth competitive examination and empathy mapping.


This project highlights the following success indicators:

Conversion Rate

A faster and simple to utilise app was created, enabling users to effortlessly add funds and pay tolls

User Engagement

Throughout the testing stage, a maximum of 2x growth in monthly active users was observed.

Customer Satisfaction

Clients expressed increased satisfaction because they believed all user needs were met.
